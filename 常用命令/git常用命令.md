Git常用命令
===============

#### 1、常用命令

- 初始化：
```
  git init
```

- 查看文件更改状态：
```
  git status
```

- 进入/返回目录
```
  cd ..
  cd directory
```

- 添加所有更改至暂存区
```
  git add ./
```

- 添加指定更改文件至暂存区
```
 git add ./derectory/xxx.java
```

- 提交暂存区至本地仓库
```
  git commit -m '提交说明'
```

- 提交所有更改内容
```
  git commit -a
```
